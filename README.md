Five console braingames built using js. First Hexlet QA Automation project.

### Installation
1. Clone the repository
2. Run `npm link` in the project directory
3. Launch one of the games using one of the following commands `brain-even`, `brain-calc`, `brain-gcd`, `brain-prime`, `brain-progression`
